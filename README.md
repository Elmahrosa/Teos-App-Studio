## 🚀 Getting Started

### Frontend (apps/web-studio)
```bash
cd apps/web-studio
npm install
npm run dev
```

### Backend (apps/api-orchestrator)
```bash
cd apps/api-orchestrator
pip install -r requirements.txt
uvicorn main:app --reload
```
